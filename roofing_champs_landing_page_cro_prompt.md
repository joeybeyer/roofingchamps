# Roofing Champs Hero Section Strategy

## Hero Section Goal

The hero section should convince a homeowner to either:

- Call now
- Start a roofing estimate request
- Request roof leak help
- Request a roof inspection
- Continue reading because the page feels trustworthy and relevant

## Hero Section Type

For Roofing Champs, use:

```text
Type 2: Key Visual on Right Side
```

Recommended desktop layout:

```text
Left side: copy, trust bullets, CTA buttons, form entry
Right side: homeowner/roofer visual
```

Why:

- Roofing is a service business.
- The user needs trust and emotional reassurance.
- A human visual works better than abstract design.
- Left-to-right reading flow supports headline → benefit → CTA.

## Avoid

Do not use:

- Abstract gradient-only hero sections
- Random AI-generated roofing scenes
- Big generic roof photos with no human context
- Complex visual clutter
- Stock photos that do not match the homeowner’s emotional state
- Copy that was written merely to fit a design

# Typeframing Method for Roofing Champs

Before designing the page, write the copy first.

Do not start with imagery, graphics, layout, or decorative elements.

## Step 1: Write the Kicker

The kicker should provide social proof, location relevance, or urgency.

Recommended kicker examples:

```text

```

```text

```

```text

```

```text

```

## Step 2: Write the Headline

The headline should clearly communicate:

- The problem
- The solution
- The benefit

Use 8–15 words when possible.

Recommended headline format:

```text
Fast Roofing Help When Your Home Can’t Wait
```

City-specific headline:

```text
Fast Roofing Help for {{City}} Homeowners
```

```text

```

Replacement-focused headline:

```text
Need Roof Repair or Replacement? Start Here.
```

## Step 3: Write the Description

The description should add detail about how Roofing Champs helps.

Use 20–40 words.

Recommended description:

```text

```

Alternative description:

```text
Tell us what is happening with your roof, share your location, and request help for repairs, inspections, storm damage, or roof replacement options.
```

## Step 4: Write the CTA Text

Use clear action language.

Primary CTA options:

```text

```

```text

```

Avoid:

```text

Continue
Learn More
```

# Hero Visual Strategy

## Use People, Not Abstract Graphics

For Roofing Champs, the key visual should usually include a real-looking person.

Best visuals:

- Roofer speaking with homeowner outside a house
- Homeowner pointing at roof damage while roofer inspects
- Roofer on ladder or near roofline, safely presented
- Homeowner looking concerned but reassured
- Roofing estimator holding clipboard/tablet
- Branded Roofing Champs truck in driveway
- Roofer inspecting shingles after storm

## Target Audience Matching

The visual should match the person visiting the page.

Target audience:

```text
Homeowners
Residential property owners
People dealing with leaks, storm damage, or roof replacement decisions
Often stressed, urgent, cautious, and looking for trust
```

Visual emotion should be:

- Concerned but reassured
- Professional
- Calm
- Helpful
- Trustworthy
- Not overly staged

## Avoid Visuals

Do not use:

- Abstract roof icons as the main hero visual
- Random AI-generated roof scenes
- People who look unrelated to roofing
- Extreme storm disaster imagery
- Dangerous roofing action shots
- Overly polished corporate stock photos
- Busy collages
- Generic city skylines as the main visual

---

# Eye Gaze Principle

Use the person in the hero visual to guide attention toward the CTA.

## Desktop Layout

Recommended:

```text
Copy and CTA on left
Person/roofer/homeowner on right
Person looking left toward headline and CTA
```

The person’s eye direction should guide the visitor back to:

1. Headline
2. Description
3. Primary CTA
4. Call button or form

## Mobile Layout

On mobile, the visual often appears below the CTA.

Recommended mobile visual:

```text
Same person or similar scene looking upward toward the CTA/form area
```

This keeps the visual guiding the visitor toward the action instead of pulling attention away.

## Gaze Prompt

Use this visual instruction:

```text
A trustworthy roofing professional standing on the right side of the hero image, looking left toward the headline and CTA area, with a concerned but reassured homeowner nearby, suburban home roof in background, professional residential roofing context.
```

Mobile-specific:

```text
A trustworthy roofing professional and homeowner shown below the CTA area, both visually angled or looking upward toward the call-to-action, clean mobile hero composition.
```

# Mobile Optimization

Mobile traffic is critical for local roofing leads.

## Mobile Rules

- H1 should be 32–44px
- Description should be 16–18px
- Buttons should stack vertically
- Tap targets should be at least 44px high
- Sticky call button should appear after initial scroll
- Form should be broken into steps
- Visual should not push CTA too far down
- Phone CTA should be visible above the fold
- Emergency leak CTA should be easy to tap

## Mobile Hero Order

Recommended mobile order:

```text
1. Kicker
2. H1
3. Description
4. Primary CTA
5. Phone CTA
6. Trust bullets
7. Form step 1
8. Visual image
```

# Design Red Flags to Avoid

Do not create landing pages that include:

- Pretty but meaningless hero images
- AI images with weird roofing scenes
- Abstract shapes replacing people
- Overly complex layouts
- No visible CTA above the fold
- Generic “Contact Us” buttons
- Long paragraphs in the hero
- No mobile-specific design
- Visual-first design where copy is forced into the layout
- Full-width roof photo with no emotional context
- Too many competing CTA buttons
- Unclear service area
- No emergency path for urgent visitors
- Fake reviews or fake local proof
