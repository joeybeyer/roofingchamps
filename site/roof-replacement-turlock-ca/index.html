When a Full System Makes More Sense

Replacement decision context: Choose replacement when heat-aged shingles, repeated leaks, and broad granule loss appear together. A full system can be the better fit when the problem is no longer isolated to one flashing point, one vent boot, or one small shingle area.

Decision fit mapping

Patch, Inspect, or Install New?

Patch

Patch fits: Choose repair when one boot, flashing point, or sun-facing shingle area explains the leak. This keeps scope narrow when surrounding materials still have service life.

Inspect

Inspection fits: Choose inspection when the first rain after dry weather reveals staining from an unclear source. This avoids replacing a system before the actual water path is confirmed.

Install new

New system fits: Choose replacement when heat-aged shingles, repeated leaks, and broad granule loss appear together. This resets more of the weather barrier instead of chasing repeated failures.
