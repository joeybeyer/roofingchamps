## Exact-Match Query Placement

Use the exact-match query only in:

1. URL
2. Title tag

Do not stuff the exact-match query into:

- H1
- Meta description
- H2s
- H3s
- Image alt text repeatedly
- Footer
- Repeated internal anchors

## Primary Exact-Match Query Format

```text
roof repair {{city}} {{state_abbr}}
```


Example:


## BERT / Search Intent Layer

Create copy that clearly answers different homeowner intents.

Cover:

### Emergency Intent

User needs help now.

Example content angle:

```text
If water is actively entering your home, start a roofing request as soon as possible. Even a small leak can damage insulation, ceilings, walls, and electrical areas.
```

### DIY vs Professional Intent

User is deciding whether to handle it themselves.

Example content angle:

```text
Homeowners may be able to spot missing shingles or ceiling stains, but roof walking, leak tracing, flashing repair, and storm damage evaluation are usually safer for roofing professionals.
```

### Repair vs Replacement Intent

User is not sure what they need.

Example content angle:

```text
A small isolated leak may only require repair, while repeated leaks, widespread shingle loss, sagging areas, or an older roof may point toward replacement.
```

### Cost Research Intent

User wants price guidance.

Example content angle:

```text
Roofing costs vary based on roof size, slope, materials, access, damage severity, decking condition, flashing, ventilation, and whether the project is a repair or full replacement.
```

### Local Provider Intent

User wants help nearby.

Example content angle:

```text
Roofing Champs helps {{City}} homeowners request roofing help for local roof issues, including leaks, storm damage, missing shingles, and inspection needs.
```
