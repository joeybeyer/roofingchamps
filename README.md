# Roofing Champs

Static Roofing Champs lead generation website.

## Local Preview

```bash
npm run build
npm run preview
```

The preview server runs at:

```text
http://localhost:4173/
```

## Cloudflare Pages

Use these settings when connecting the GitHub repo:

```text
Framework preset: None
Build command: npm run build
Build output directory: site
Root directory: /
```

The canonical domain in generated schema is:

```text
https://www.roofingchamps.net
```
